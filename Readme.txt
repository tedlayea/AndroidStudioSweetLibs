Instructions for Android Studio Installation
1. Download from latest Android Studio from https://developer.android.com/studio/index.html#top
2. Extract the download file and move the extracted folder to where you want to install the program
3. In terminal, open the above folder, then open bin folder
4. In the terminal, run: sudo chmod 777 -R studio.sh
5. Enter your password
6.  In the terminal, run: ./studio.sh

//Add the following in AndroidManifest.xml to access the internet in our application.
<uses-permission android:name="android.permission.INTERNET" />
